Manual

Introduction

Welcome to Xtractify, an AI-powered SaaS application that simplifies the extraction of key details from invoice PDFs. This manual will guide you through the features, installation, and usage of Xtractify.

Getting Started

1. Installation

Follow the steps outlined in the Installation section of the README to set up the application on your local machine.

2. Running the Application

Start the application using the command:

node server.js

Open your web browser and navigate to http://localhost:3000.

3. Uploading an Invoice

Step 1: Click the "Choose File" button on the homepage.
Step 2: Select the invoice PDF from your computer.
Step 3: Click "Upload" to process the PDF.

4. Displaying the extracted information

After the processing is complete, it will display details containing the extracted information.

Advanced Usage

API Key Configuration

You can configure the API keys used by Xtractify in the .env file. Ensure that you have valid keys for both the Google Gemini API and OpenAI.

Customization

If you want to customize the extraction process or the output format, you can modify the giveDetails function in the index.js file.

Deployment

To deploy Xtractify to a cloud service:

  • Ensure all environment variables are correctly set up in the deployment environment.
  • Configure the server to run the server.js script.
  • Adjust the port variable in server.js if needed for the cloud environment.
